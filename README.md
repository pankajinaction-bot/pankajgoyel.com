# PankajGoyel.com

Static website for Pankaj Goyel.

## Files

- `index.html` — website structure and text
- `style.css` — visual styling
- `assets/` — images and logos
- `netlify.toml` — Netlify configuration
- `_redirects` — redirect fallback
- `robots.txt` — search indexing instructions
- `sitemap.xml` — basic sitemap

## Publish with GitHub + Netlify

1. Create a new GitHub repository, for example: `pankajgoyel.com`
2. Upload all files from this folder to the repository.
3. Open Netlify.
4. Choose **Add new site > Import an existing project**.
5. Connect GitHub and select the repository.
6. Build command: leave empty.
7. Publish directory: leave as `/` or set to `.`.
8. Deploy site.
9. In Netlify, go to **Domain management** and add:
   - `pankajgoyel.com`
   - `www.pankajgoyel.com`
10. Update your domain DNS at GoDaddy/registrar as Netlify instructs.
